Habia una vez un patito que decia miau miau, este patito queria aprender a utilizar diferentes tipos de formulario.
Existen dos metodos principales para la obtencion de los datos dentro de un formulario web, para lo cual aqui lo explicaremos:


Get:

Este metodo es el estandar como salida del navegador mediante el cual se resuelve una peticion por parte del cliente, asi mismo en la salida estandar(url), se puedene visualizar los datos enviados mediante el formulario.

POST:

Este metodo es el mas utilizado para el envio de informacion desde un formulario atendiendo una peticion por parte del cliente, actuando para poder agregar o insertar nuevos datos en una BD, asi mismo a diferencia del metodo GET este utiliza "ubn tipo de seguridad ", mediante el cual la informacion no es visible por medio de la salida estandar del navegador.
